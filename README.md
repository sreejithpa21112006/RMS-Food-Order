# RMS-Food-Order

A full-stack, cloud-ready Restaurant Management System (RMS) built with Flask, Firebase, and TiDB.

## Features
- **Firebase Authentication**: Secure login system with Google Sign-in integration.
- **Role-Based Access Control**: Separate, secure dashboards for Admins, Kitchen Staff, and Front Staff.
- **Kitchen Display System (KDS)**: Real-time ticket board for chefs to mark orders as Preparing or Ready.
- **Dynamic Menu Management**: Admins can add, edit, and classify menu items.
- **Cloud-Ready**: Configured for deployment on Render and TiDB Serverless.
